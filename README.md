# colour-package-python-
A easy to use python package to change colour  output on terminal



# Supported Colors:
BLACK 
RED 
GREEN 
YELLOW 
BLUE
MAGENTA 
CYAN 
WHITE 
UNDERLINE
RESET 
------------------------------------------

# Usage:
from color import s
print(s.RED+'ERROR')

Easy Use: [eu]
from color import eu
print(eu.R+'Error')
